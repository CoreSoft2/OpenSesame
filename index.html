JFX-OpenVPN-Client by jfx2006

Yet another OpenVPN GUI for "roadwarriors".

This is yet another GUI for OpenVPN. For now I'm targeting, Windows, as thats what my immediate need is. I have however successfully run the GUI on Linux and Mac OSX. The idea is to be able to make connections to OpenVPN servers, and maintain configurations without having to resort to text editors and command lines. As soon as I get a decent build going for Windows, I will post an installer in the downloads section. My plan is to install everything you need to make this work from one package... the GUI, plus OpenVPN and its dependencies.

Dependencies

There will be no dependencies if you install with the WIndows installer (not available yet). It will include everything you need. To run from source, like on Linux you will need: PyQt 4.5 and its dependencies - SIP 4.8 - Qt 4.5 OpenVPN and its dependencies - TUN/TAP kernel driver - OpenSSL - LZO - PKCS11-helper
